VisualBinary
============

VisualBinary is a study project for 3D binary visualization


Hotkeys
----

**Camera :**
*********

MouseWheel, PageUp, PageDown : Zoom

Up, Down, Left, Right : Rotate camera

X : Stop auto-rotating

C : Enable auto-rotating

**Offsets :**
**********

+, - : Change start offset

*, / : Change end offset


**Mode :**
************

S : Space view

T : Time view
